PROJECT SUMMARY:

Create a mobile application from existing Knowing-Gaia webpage (http://knowing-gaia.net) using design concepts from study-groups. App should include the html semantic tags including article and section tags, and should meet accessibility standards. App will have navbar functionality that is simple and easy to use and be created using html, CSS and some JS if necessary. No CSS framewords will be used.
